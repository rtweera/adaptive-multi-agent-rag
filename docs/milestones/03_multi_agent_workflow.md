# Milestone 03 — Multi-Agent Workflow

## Goal

Transform the system from a simple RAG pipeline into an orchestrated multi-agent platform.

---

# Agents

## Query Understanding Agent

Responsibilities:

- query rewriting
- intent extraction
- retrieval strategy selection

---

## Retrieval Agent

Responsibilities:

- vector retrieval
- keyword retrieval
- reranking

---

## Response Agent

Responsibilities:

- grounded synthesis
- citation-aware responses
- final answer generation

---

# LangGraph

Implement:

- workflow graph
- agent transitions
- conditional routing
- execution traces

---

# UI Additions

Display:

- workflow execution path
- active agent
- retrieval trace

---

# Success Criteria

The system should:

- visibly orchestrate multiple agents
- expose workflow traces
- improve response quality
